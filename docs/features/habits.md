# Habits Module

The Habits module helps you build and maintain positive daily habits through tracking, streaks, and statistics.

## Features

- Daily habit tracking
- Streak counting
- Statistics and charts
- Gamification integration
- Reminders

## Creating Habits

1. Press `Ctrl+3` to open Habits
2. Click "+ New Habit"
3. Enter habit name (e.g., "Exercise", "Read", "Meditate")
4. Click "Add"

### Habit Properties

| Property | Description |
|----------|-------------|
| Name | Habit identifier |
| Streak | Consecutive days completed |
| Total | All-time completions |
| Created | When habit was added |

## Daily Tracking

### Marking Complete

1. Click the checkbox next to a habit
2. The habit is marked done for today
3. Streak increments if previous day was done

### Best Time to Track

Track habits at the same time each day:
- Morning review
- Evening reflection
- After completing the action

## Streaks

Streaks motivate consistency:

| Streak | Effect |
|--------|--------|
| 1-6 days | Building momentum |
| 7+ days | Weekly streak badge |
| 30+ days | Monthly streak badge |
| 100+ days | Century achievement |

### Streak Protection

Missing a day resets the streak to 0. Plan ahead:
- Track habits early
- Set reminders
- Have backup plans

## Statistics

View your habit data:

- **Weekly view**: Last 7 days completion
- **Monthly view**: Calendar heat map
- **Completion rate**: Percentage of days completed
- **Best streak**: Longest consecutive streak

## Gamification

Habits contribute to your XP:

| Action | XP |
|--------|-----|
| Complete habit | +10 XP |
| Streak bonus (7+ days) | +5 XP extra |
| Streak bonus (30+ days) | +10 XP extra |

### Achievements

Unlock achievements for habit milestones:
- First habit tracked
- 7-day streak
- 30-day streak
- 100-day streak

## Best Practices

### Starting New Habits

1. **Start small** - "Read 1 page" not "Read 1 hour"
2. **Be specific** - "Walk 10 minutes" not "Exercise"
3. **Habit stack** - Attach to existing routines
4. **One at a time** - Add new habits gradually

### Maintaining Habits

1. **Track immediately** - Don't delay
2. **Review weekly** - Check your stats
3. **Forgive misses** - Don't let one day derail you
4. **Celebrate streaks** - Acknowledge progress

### Choosing Habits

Good habits for productivity:
- Morning routine
- Exercise/movement
- Reading/learning
- Journaling
- Focused work blocks
- Evening review
