## What is BytePad?

BytePad is a keyboard-first productivity application designed for developers and power users. It combines notes, tasks, habits, journaling, and AI assistance in a unified, Notepad++ inspired interface.

### Key Features

| Feature | Description |
|---------|-------------|
| Notes | Markdown editor with backlinks and wikilink support |
| Tasks | Priority-based task management with subtasks and calendar view |
| Habits | Daily habit tracking with streaks and statistics |
| Journal | Daily notes with mood and energy tracking |
| Focus Mode | Pomodoro timer with task integration |
| FlowBot | AI-powered productivity coach |
| Knowledge Graph | Visual connections between notes, tasks, and bookmarks |
| Gamification | XP, levels, and achievements system |
