# Contributing Guide

See [CONTRIBUTING.md](../../CONTRIBUTING.md) in the root directory for full guidelines.

## Quick Start

```bash
# Clone and install
git clone https://github.com/samitugal/bytepad.git
cd bytepad
npm install

# Development
npm run dev          # Web
npm run dev:electron # Desktop
```

## Code Style

- TypeScript strict mode
- Functional React components
- Tailwind CSS for styling
- Components < 300 lines
