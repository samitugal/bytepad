### Windows Installation Note

The Windows installer is code-signed. If you still see a SmartScreen warning:
1. Click **"More info"**
2. Click **"Run anyway"**

This is normal for new applications building reputation with Microsoft SmartScreen.

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Run Electron app in development
npm run dev:electron

# Build for production
npm run build:electron

# Package for distribution
npm run package
```

## Tech Stack

- **Frontend**: React 18, TypeScript, TailwindCSS
- **Desktop**: Electron
- **State Management**: Zustand
- **AI Integration**: OpenAI, Anthropic, Google AI, Groq, Ollama
